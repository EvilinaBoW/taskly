# Taskly MVP 

Локальный трекер задач для малого бизнеса и фрилансеров.

## Запуск
1. Скачайте файл `taskly.html`
2. Откройте его в любом браузере (Chrome/Firefox/Edge)
3. Все данные сохраняются автоматически в localStorage вашего браузера.

## Технологии
- Чистый HTML5 + CSS + JS
- Полностью оффлайн
